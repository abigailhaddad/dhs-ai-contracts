Overview

This analysis identifies DHS contracts that involve artificial intelligence or machine learning work. Contract descriptions were searched for — AI-related keywords, then each matching contract was reviewed by an LLM to filter out false positives (e.g., fingerprint scanners labeled "biometric," or generic "algorithm" references in non-AI work).

Methodology

Data source

Contract data comes from the USASpending.gov API, which publishes all federal contract awards. This analysis covers DHS contracts awarded between FY2022 (October 1, 2021) and FY2026 (September 30, 2026). Only prime contract awards are included — not grants, IDV vehicles, or subawards.

Keyword search

Contract descriptions were searched for the following terms. All terms were included regardless of whether they returned results — zero-hit terms are shown to demonstrate search coverage.

Broad terms like algorithm, biometric, and decision support were kept intentionally — the LLM pass handles false positives.

LLM classification

Each keyword-matched contract was passed to GPT-5.4-mini with the contract description, vendor, sub-agency, NAICS code, and PSC code. The model was asked whether the contract is primarily for AI or machine learning work, and to provide a one-sentence explanation.

The classifier was instructed to lean toward true for borderline cases — this analysis aims to be comprehensive rather than conservative.

Caveats

  • This captures only contracts whose descriptions mention AI-related terms. AI work embedded in larger IT contracts without explicit AI language will be missed.
  • Dollar amounts are total obligated value and may include option years not yet exercised.
  • The same work may appear as multiple contract actions; deduplication is by USASpending Award ID.
  • DHS sub-agencies include CBP, ICE, TSA, USCIS, FEMA, Coast Guard, Secret Service, and others.
