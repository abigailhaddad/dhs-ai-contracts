Classification review

Spot-check the LLM's AI/ML verdict for every contract. Reviews are stored in your browser and exportable as CSV.
How to use this: for each row, mark the LLM's verdict (currently always yes / AI since this dataset only ships is_ai = true) as agree, disagree, or skip. Add a note if you want. Reviews persist in this browser via localStorage; nothing is sent anywhere. Click Export CSV to share or commit your review file.
Workflow: use the filter bar to cut down to the source path or sub-agency you want to audit, then walk the rows. The review CSV mirrors the pattern in pull_usaspending/crosswalk_review.html — drop the export into data/classification_reviews/ when you're done.
